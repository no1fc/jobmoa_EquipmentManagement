# 개발 진행 현황 추적 규칙

## 문서 업데이트 타이밍

### 개발 시작 시 (Phase/단계 착수)
새로운 Phase 또는 단계(A1, B2, C5 등)의 개발을 시작할 때:
1. `docs/progress.md`에서 해당 단계의 상태를 `🔨 진행중`으로 변경
2. Notion 페이지(34f2af5a-9ceb-8174-8067-c0753266d8fc)의 개발 진행 현황 테이블에서 해당 단계 상태를 업데이트

### 개발 완료 시 (Phase/단계 완료)
Phase 또는 단계의 개발이 완료될 때:
1. `docs/progress.md`에서 해당 단계의 상태를 `✅ 완료`로 변경하고 완료일, 주요 산출물 기록
2. 전체 진행률 퍼센테이지 업데이트
3. Notion 페이지의 개발 진행 현황 테이블에서 해당 단계 상태를 ✅ 완료로 변경
4. Notion 페이지의 MVP 범위 체크리스트에서 관련 항목 체크
5. Notion 페이지의 보완 필요 사항에서 해결된 항목 체크
